# سیستەمی خشتەی هەفتانەی قوتابخانە — وەشانی دێسکتۆپ

بەرنامەیەکی ڕەسەنی ویندۆز کە **بەبێ XAMPP** کار دەکات. هەموو شتێک (سێرڤەری PHP، بنکەی زانیاری) لە ناو خودی بەرنامەکەدایە.

## جیاوازی لەگەڵ وەشانی XAMPP
- **پێویستی بە XAMPP نییە** — تەنها بەرنامەکە دابمەزرێنە و بیکەرەوە
- بنکەی زانیاری SQLite ـە (فایلێکی بچووک بە ناوی `timetable.db` لە تەنیشت بەرنامەکە)، نەک MySQL
- یەکەم جار کە دەیکەیتەوە، بنکەی زانیارییەکە خۆکارانە دروست دەبێت
- هەموو تایبەتمەندییەکان هەر ئەوانەن: ڕێگری تێکهەڵچوون، میلاک، ڕۆژە بەتاڵەکان، پێشنیاری ئۆتۆماتیک

## دامەزراندن
دوو ڕێگە هەیە، هەر کامەیان بتەوێت:

**١. دامەزرێنەر (Setup.exe):**
فایلی `TimetableSystem-Setup.exe` دابگرە و دووجار کرتەی لێ بکە. وەک هەر بەرنامەیەکی ویندۆز دادەمەزرێت و ئایکۆنێک لەسەر Desktop دادەنێت.

**٢. پۆرتەبڵ (بێ دامەزراندن):**
فایلی `TimetableSystem-Portable.zip` بکەرەوە و فایلی `TimetableSystem.exe` ـەکەی ناوی دووجار کرتەی لێ بکە. هیچ دامەزراندنێک ناوێت — تەنانەت لەسەر فلاش میمۆریش کار دەکات.

## بەکارهێنان
هەر وەک وەشانی وێبەکە:
١. مامۆستایان، بابەتەکان، و پۆلەکان زیاد بکە
٢. لە تابی مامۆستایان، دوگمەی 🗓 بۆ دیاریکردنی ڕۆژە بەتاڵەکان
٣. لە تابی خشتە، پۆلێک هەڵبژێرە و کرتە لە خانەکان بکە بۆ دانانی وانە
٤. سیستەمەکە ڕێگری لە تێکهەڵچوون دەکات و پێشنیاری مامۆستای بەردەستت بۆ دەکات

## پاراستنی داتا (Backup)
هەموو داتاکەت لە فایلی `timetable.db` دایە. بۆ پاڵپشتی، تەنها ئەو فایلە کۆپی بکە بۆ شوێنێکی سەلامەت. بۆ گەڕاندنەوە، فایلەکە بگەڕێنەوە شوێنەکەی.

## چۆن ئەم فایلانە دروست بوون؟
ئەم فایلانە بە GitHub Actions دروست دەبن (لەسەر کۆمپیوتەری ویندۆز لە هەورەوە). بۆ دروستکردنەوەیان دوای گۆڕانکاری، تەنها کۆدەکە بۆ GitHub بنێرە و خۆی فایلە نوێیەکان دروست دەکات لە فۆڵدەری `release`.
